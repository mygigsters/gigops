# Changelog

All notable changes to GigOps will be documented in this file.

## [0.1.0] — 2026-04-18

### Added
- **Gig evaluation** — AI-powered 6-dimension scoring (A–F grades) for any gig URL
- **Proposal generation** — Tailored, ready-to-paste proposals from your profile
- **Platform scanning** — Saved searches for Airtasker and Freelancer
- **Client intelligence** — Scrape and analyse client profiles, flag red flags
- **Pipeline tracking** — JSONL-based gig pipeline with terminal dashboard (Go)
- **Rate benchmarking** — Community rate data with anonymous opt-in sharing
- **Multi-LLM support** — Anthropic, OpenAI, Google Gemini, Ollama (local)
- **Config CLI** — `gigops config show/get/set` for managing preferences
- Platform support: Airtasker, Freelancer, Upwork (limited — Cloudflare), any URL
