Get started

Three commands to freedom

01

Install

One command. No accounts, no sign-ups, no credit cards.

npm install -g gigops
02

Configure

Point it at your preferred LLM — Claude, GPT, Gemini, or run it fully local with Ollama.

export ANTHROPIC_API_KEY="sk-..."
03

Evaluate

Paste a gig URL and get an instant AI-powered breakdown. Bid smart, not blind.

gigops evaluate <url>
